Scope & Limitation

I. Objective:

  • The system required log in and log out process
  • The system can add/edit/delete employee’s record
  • The system has a capability to search employee’s record and attendance record
  • The system provide auto generate a unique QR Code for each employee
  • The system provide auto record for daily’s/weekly’s/monthly’s attendance in a excel sheet
  • The system has a capability to store all deleted employee’s record in history section
  • The system provide a system log for track/trace purposes

II. Scope

1.0 Security Module

  • Log-in & Log-out – the system provide a username and password before can use/access the program

2.0 Manage Employee Record Module

  • Add/Edit/Delete Employee’s Record – this system has a capability to add new employee record, can modified the information record, and also can delete the record(Resign, Got Fired)

3.0 Auto Generate Module

  • View Generated QR Code – this system provide to store all unique QR Code in the system and the user/client can views for qr each code that have a label/file name of employee
  • Daily/Weekly/Monthly Attendance – this system provide to display all attendance record of each employee

4.0 Search Module

  • Search Bar – this system provide to search an employee record and attendance record

5.0 History Module

  • History – this system has a capability to store all recently deleted files/employee’s record Resign, Got Fired)
  • System Log – this system provide records for all activities happened in the system

III. Limitation:

  • Only for daily’s attendance
  • No internet access
  • No calculation for daily income
  • Limited user(Only for the HR and the Stakeholder)
  • Only basic information of employee’s record available
  • Support only in a computer device
