Project Charter

Project Title: Employee Attendance System by QR Scan

Project Start Date: June 1, 2019

Project Finish Date: October 27, 2019

Budget Information: This system has allocated PHP 50,000. 00 for this project. The majority of cost for this project will be on usage of the internet (Data usage).

Project Manager: Timon, Junmark M, +639267978924, junmarktimon@gmail.com

Project Objectives:

  • The system required log in and log out process
  • The system can add/edit/delete employee’s record
  • The system has a capability to search employee’s record and attendance record
  • The system provide auto generate a unique QR Code for each employee
  • The system provide auto record for daily’s/weekly’s/monthly’s attendance in a excel sheet
  • The system has a capability to store all deleted employee’s record in history section
  • The system provide a system log for track/trace purposes

Approach:

We gathered other related sources to make this project well. This system required an authentication for security purposes. The other features on this system having a limited to manipulate the data. Auto generate for a QR code and other reports and the admin/user can trace using the history log features to monitor what happened on this system.
